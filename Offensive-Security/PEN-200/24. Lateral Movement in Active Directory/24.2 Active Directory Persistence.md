### Golden Ticket
#goldenticket 
Retrieve krbtgt password hash to create TGTs (golden tickets)
Gives us permissions to access entire domain's resources

#### Pre-req
- Access to Domain Admin's group account
- Compromised DC itself

#### Scenario
RDP to DC as _jeffadmin_.

Execute mimikatz with `lsadump::lsa` to dump krbtgt password hash

Abuse `vshadow.exe` to create shadow copy of **ntds.dit** credential database

RDP as _jeffadmin_ to DC1. 
Launch elevated command prompt.
Execute `vshadow.exe` `-nw`- disable writers/speed up backup creation and `-p`- store copy on disk
